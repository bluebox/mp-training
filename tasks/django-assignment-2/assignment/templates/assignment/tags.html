tags are used inside {%%} tags

block

Defines a block that can be overridden by child templates

syntax ->

{% block 'block_name' %}

{% endblock %}

comment

Ignores everything between {% comment %} and {% endcomment %}. An optional note may be inserted in the first tag. For example, this is useful when commenting out code for documenting why the code was disabled, check exmaple in code

syntax ->

{% comment 'comment_name' %}

{% endcomment %}

{% comment %} This is a comment, this lines wont be seen in webpage {% endcomment %}

IF ELSE

syntax ->

{% if 'condition_name' %}

{% endif %}

{% if age > 18 %}

major student

{% else %}

minor student

{% endif %}

FOR IN

syntax ->

{% for in list%}

{% endfor %}

{% for value in hobbies %}
